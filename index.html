Ray-Ban Meta Voice Commands

Speak these commands hands-free. Audio plays through your glasses speakers. Say commands clearly — no wake word needed.

💕 Swiping
"like"
Like the current profile (also: "yes", "love", "heart")
"pass"
Pass on the current profile (also: "nope", "no", "next", "skip")
"super like"
Send a super like to stand out (also: "super")
📖 Learn More
"read bio"
Hear their full bio read aloud (also: "tell me about them")
"interests"
Hear what they're into (also: "hobbies", "what do they like")
"how old"
Hear their age and location (also: "age", "where are they")
"open profile"
Expand the full profile view (also: "more info", "expand")
🧭 Navigation
"discover"
Go to the swipe/discover feed (also: "swipe", "browse", "home")
"matches"
Go to your matches list
"messages"
Go to your conversations (also: "inbox", "chat")
"my profile"
View your own profile
💬 Messaging (while in chat)
"send [message]"
Sends that text — e.g. "send hey, how's your day going?"
🔧 Controls
"status"
Hear your match count and unread messages
"stop"
Stop the current speech (also: "mute", "quiet")
"help"
Open this help screen
"close"
Close this help screen (also: "dismiss", "got it")

Connect Firebase

Follow these steps to make Spark work with real users

Step 1 — Create a Firebase Project
Go to console.firebase.google.com → Add project → give it a name (e.g. "spark-dating") → Continue (disable Analytics if you like) → Create project.
Step 2 — Enable Authentication
In your project → Build › Authentication → Get started → Sign-in method → Email/Password → Enable → Save.
Step 3 — Enable Firestore Database
Build › Firestore Database → Create database → Start in test mode → choose your region → Enable.
Step 4 — Enable Storage
Build › Storage → Get started → Start in test mode → Done.
Step 5 — Get Your Config
Project Settings (gear icon) › General → scroll to Your apps → click </> (Web) → Register app → copy the firebaseConfig object.
Step 6 — Paste Config Below & Save
Open spark-dating.html in a text editor, find FIREBASE_CONFIG near the top of the <script> section, and replace the placeholder values with yours. Or paste your config here:
